# MCP PlantUML for Ampcode

Integration of Model Context Protocol (MCP) with PlantUML for diagram generation in Ampcode.

## Generated Diagrams

- **MCP Registration**: Sequence diagram showing the initialization and interaction with the PlantUML MCP server.
- **NATS JetStream Architecture**: Component diagram for NATS JetStream integration.
- **Sketchbook Architecture**: Diagram illustrating the sketchbook feature design.
